\section{Introduction}
This is an image retargeting program using Seam Carving\cite{sc} algorithm.

\subsection{Compilation}
Dependencies:

\begin{enumerate}
    \item gcc >= 4.7
    \item GNU make
    \item Magick++
\end{enumerate}

Compilation:
\begin{lstlisting}
$ make
\end{lstlisting}
